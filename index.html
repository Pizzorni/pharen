Pharen: Lisp -> PHP

Pharen is a Lisp that compiles to PHP, bringing lexical closures, macros, less PHP, and more.

{% highlight clojure %} (fn greet-person (name) (. "Hello " name "!")) (prn (greet-person "visitor")) ; => Hello a visitor! {% endhighlight %}
